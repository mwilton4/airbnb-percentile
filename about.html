About This Project

Problem: What does a 4.7/5 rating mean for an Airbnb in Amsterdam? This project is designed to provide users with context of how an Airbnb compares to others in the same city. Airbnb listings are rated on a scale of 1 to 5, but without knowing the distribution of ratings, it can be difficult to interpret what a specific rating means for the quality of the listing.

Features

  • Interactive visualization of rating distributions for Airbnb listings.
  • City-specific percentile calculations for review scores.
  • User-friendly interface with dropdown city selection and percentile results.

Purpose and Value

This project demonstrates my skills in web development, data analysis, and visualization. By combining R for data preprocessing and JavaScript for interactivity, I created a tool that is both functional and user-focused. It showcases my ability to:

  • Analyze and clean real-world datasets.
  • Develop dynamic web applications using HTML, CSS, and JavaScript.
  • Leverage libraries like Chart.js for interactive data visualization.
  • Deploy projects for public access using platforms like GitHub Pages.

Technical Details

Tech Stack: R, JavaScript, HTML, CSS, Chart.js, GitHub Pages
Key Libraries: jsonlite (R), Chart.js (JavaScript)
Data Sources: Simulated Airbnb data formatted in CSV and converted to JSON.

Future Enhancements

Planned updates include expanding city coverage, integrating live data APIs, and adding additional metrics such as price distributions and booking trends.
